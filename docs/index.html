🇻🇪 RescueGIS

Software libre que convierte reportes ciudadanos en una cola de rescate priorizada por edificio.

Cómo funciona

ciudadano (WhatsApp/Telegram/web) ──► matching GPS → edificio OSM (radio adaptativo según precisión; nunca fuerza matches inciertos) ──► correlación de reportes duplicados (posición refinada + confianza multi-fuente) ──► cola priorizada (fórmula transparente) ──► API + mapa del coordinador ──► teléfono del rescatista (navegar / llamar)

Qué está funcionando ya

  • Bot de Telegram: reporte ciudadano en ~30 segundos con GPS
  • Conector ChatMap (HOT): los chats de WhatsApp mapeados con la herramienta oficial de la activación entran directo a la cola
  • Conector SOS Venezuela 2026: federación respetando su privacidad de coordenadas
  • API REST con roles (público degradado / cuerpos de emergencia con precisión total), exportes GeoJSON/CSV para QGIS
  • Vista móvil de campo y push Telegram para rescatistas en terreno
  • Panel de coordinación con mapa Leaflet en tiempo real

Se busca

  • 👨‍💻 Programadores Python / PostGIS / JS — preséntate aquí
  • 🤝 Integraciones con plataformas de la respuesta (HOT, plataformas ciudadanas, cuerpos de emergencia)
